# 获取自己的微信号

**简要描述：**

* 获取自己的微信号

**请求URL：**

* 开发者域名 +`/self`

**请求方式：**

* POST

**参数：**

| 参数名 | 必选 | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| wId | 是 | string | 微信实例ID |

**返回数据：**

| 参数名 | 类型 | 说明 |
| :--- | :--- | :--- |
| code | int | 1000成功，10001失败 |
| msg | string | 反馈信息 |
| data |  |  |
| wcId | string | 微信号 |

**请求参数示例**

```text
{
    "wId": "0000016e-6343-089e-0001-e2ef939176f6"
}
```

**成功返回示例**

```text
{
    "message": "成功",
    "code": "1000",
    "data": {
        "wcId": "wxid_6mq1pu8ngj3r22"
    }
}
```

**错误返回示例**

```text
{
    "message": "失败",
    "code": "1001",
    "data": null
}
```
