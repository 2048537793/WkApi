# 下载文件

**请求URL：**

* http://域名地址/getMsgFile

**请求方式：**

* POST

**请求头Headers：**

* Content-Type：application/json

**参数：**

| 参数名 | 必选 | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| wId | 是 | string | 微信实例ID  包含此参数 所有参数都是从消息回调中取） |
| msgId | 是 | long | 消息id |
| from\_user | 是 | string | 发送人的微信号 |
| to\_user | 是 | string | 接收人的微信号 |
| content | 是 | string | 收到的消息的xml数据 |

**返回数据：**

| 参数名 | 类型 | 说明 |
| :--- | :--- | :--- |
| code | int | 1000成功，10001失败 |
| msg | string | 反馈信息 |
