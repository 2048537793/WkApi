# 发送已经收到的链接消息

**简要描述：**

* 根据回调收到的xml content信息发送链接消息

**请求URL：**

* `http://域名地址/forwardUrl`

**请求方式：**

* POST 

**请求头Headers：**

* Content-Type：application/json
* Authorization：login接口返回

**参数：**

| 参数名 | 必选 | 类型 | 说明 |
| :---: | :---: | :---: | :---: |
| wId | 是 | string | 微信实例ID |
| wcId | 是 | string | 微信号/群号 |
| content | 是 | string | xml视频内容 |

**返回数据：**

| 参数名 | 类型 | 说明 |
| :---: | :---: | :---: |
| code | int | 1000成功，10001失败 |
| msg | string | 反馈信息 |
